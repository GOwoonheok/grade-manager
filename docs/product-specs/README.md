# product-specs — 기능 명세

기능별 1문서. 새 기능 구현 **전** 명세를 쓰고, 변경 시 갱신.
작성 틀: 사용자 시나리오 → 동작 정의 → 엣지 케이스 → 관련 코드 위치.

## 현재 구현된 기능 (문서화는 점진적)
- 성적: 다과목·수강등록, 가중치 최종점수, 상대평가 등급(동점 올림, 0001 제외), 성적 공개 토글
- 답안지: 과목단위 사진 업로드/뷰어(비공개 버킷)
- 플래시카드 학습: 분야(deck)→토픽→카드, 모드(순서/랜덤/체크만), 섹션 점진공개·가리기·★체크·글자크기
- 플래시카드 관리: 과목·토픽·카드 CRUD, 엑셀 업/다운, 이미지, 순서변경, 학습신청 승인
- Q&A(같이 공부하기), 전화 암호화
