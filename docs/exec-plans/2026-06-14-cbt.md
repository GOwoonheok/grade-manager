# 2026-06-14 — CBT 예상문제 (C1)

## 목표
4지선다 CBT — 교수가 예상문제 등록·검수, 학생이 풀고 채점·해설 확인. 무지출(Supabase 텍스트).

## 범위
- **포함(C1)**: `quiz_questions`/`quiz_attempts`(017) + RLS, `QuizAdmin`(등록·검수·상태), `QuizPlayer`(분야/토픽·문항수 → 풀이 → 채점 → 오답·해설 리뷰).
- **제외**: AI 자동생성(C2 — `/api` 백엔드), 오답노트·통계·타이머(C3).

## 설계 메모
- 문제 풀 노출은 `status` 로 통제: 교수 직접 등록 = `verified` 즉시, (C2 AI 생성분) = `draft` → 검수 후 verified. 학생은 RLS로 verified만.
- `getQuizSet`은 verified만 섞어 N개. 응시 결과는 `quiz_attempts`(detail jsonb)로 저장 → C3 오답노트 기반.
- 컴포넌트는 017 미적용 시 화면 안 깨지게 friendly 오류 처리(016 사고 교훈).

## 결과 (완료 2026-06-14)
- 파일: `supabase/017_quiz.sql`, `src/lib/quiz.ts`, `src/components/QuizAdmin.tsx`, `src/components/QuizPlayer.tsx`, 학습 메뉴 연결(학생=CBT 문제풀이 / 교수=CBT 문제 관리).
- `npm run check` + `npm run build` 통과. main 배포.
- **운영 필수**: Supabase SQL Editor에서 **017 실행**.

## 후속
- C2: `/api/quiz-gen`(Gemini 무료, 토픽 카드 컨텍스트로 4지선다 생성) → draft → 교수 검수 → verified.
- C3: 오답노트·통계·타이머.
